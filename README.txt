## Setup instructions
- Clone the repository: git clone https://github.com/adi-tomar/Housing_Weather_UL_Dim_Reduction/
- Navigate to the project directory: Housing_Weather_UL_Dim_Reduction
- Create a virtual environment python -m venv venv source venv/bin/activate
- Install dependencies: pip install -r requirements.txt
## Running the algorithms and plots
- algorithms/UL_weather.ipynb
- algorithms/UL_housing.ipynb
## Data
Weather data is located in data/weather/city_data
Housing data is located in data/housing
